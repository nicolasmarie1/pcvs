pcvs.backend.bank module

class pcvs.backend.bank.Bank(token)[source]

Bases: Bank

Representation of a PCVS result datastore.

Stored as a Git repo, a bank hold multiple results to be scanned and used to analyse benchmarks result over time. A single bank can manipulate namespaces (referred as ‘projects’). The namespace is provided by prefixing proj@ to the bank name.

Build a Bank.

A Bank is describe by an optional token follow by a bank name or bank path.

Example: cholesky@mpc_ci_bank, nas@/home/mpc/mpc_ci_bank

Parameters:

token (str) – name, path, project@name or project@path
