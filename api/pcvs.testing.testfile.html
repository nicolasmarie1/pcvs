pcvs.testing.testfile module

class pcvs.testing.testfile.TestFile(file_in, path_out, data=None, label=None, prefix=None)[source]

Bases: object

A TestFile manipulates source files to be processed as benchmarks (pcvs.yml & pcvs.setup).

It handles global information about source imports & building one execution script (list_of_tests.sh) per input file.

Variables:

  • _in (str) – YAML input file

  • _path_out (str) – prefix where to store output artifacts

  • _raw (dict) – stream to populate the TestFile (rather than opening input file)

  • _label (str) – label the test file comes from

  • _prefix (str) – subtree the test file has been extracted

  • _tests (list) – list of tests handled by this file

  • _debug (dict) – debug instructions (concatenation of TE debug infos)

Constructor method.

Parameters:

  • file_in (str) – input file

  • path_out (str) – prefix to store artifacts

  • data (dict[str, Any] | None) – raw data to inject instead of opening input file

  • label (str | None) – label the TE is coming from

  • prefix (str | None) – testfile Subtree (may be Nonetype)
