pcvs.orchestration.manager module

class pcvs.orchestration.manager.Manager(max_nodes=0, publisher=None)[source]

Bases: object

Gather and manipulate Jobs under a hiararchical architecture.

A Manager is in charge of carrying jobs during the scheduling. Jobs are divided hierarchicaly (basically, depending on the number of resource requested to be run). To extract jobs to be scheduled, a Manager will create Set()s. Once completed Sets are merged to the Manager before publishing the results.

Initialize a Job Manager.

Parameters:

  • max_nodes (int) – max number of nodes allowed to schedule.

  • publisher (ResultFileManager) – requested publisher by the orchestrator
