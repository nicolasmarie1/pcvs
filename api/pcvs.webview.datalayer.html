pcvs.webview.datalayer module

class pcvs.webview.datalayer.DataRepresentation[source]

Bases: object

Data Manager from the Flask server side.

This class manages insertion & requests to the data tree gathering test results from a single report server.

the data representation looks like:

  0:
fs-tree:
    <LABEL1>:
        __metadata: {<counts>}
        __elems:
            <subtree1>:
                __metadata: {<counts>}
                __elems:
                    <te1>:
                        _metadata: {<counts>}
                        _elems: [Test(), Test(), Test()]
tags:
    <tag1>:
        __metadata: {<counts>}
        __elems: [Test(), Test(), Test()]
iter:
    <it_name>:
        __metadata: {<counts>}
        __elems:
             <possible_value>:
                 __metadata: {<counts>}
                 __elems: [Test(), Test(), Test()]
failures: [ Test(), Test(), Test()]
# ...
