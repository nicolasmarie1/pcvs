pcvs.analysis.graph module

Graph module.

Use analysis to query data from bank and use theme to create mathplotlib graphs. Used by cli_graph to draw graph from the command line.

2 types of graphs:

  • get_status_series: a stacked graph representing the success rate over time of a series of run where each test status represent a layer.

  • get_time_series: a graph for each basetest (test without criterions) in witch we represent the test duration evolution per run, one line for each criterions combination.

pcvs.analysis.graph.get_status_series(analysis, series, path, show, extension, limit)[source]

get_status_series: create a test state graph.

Parameters:

  • analysis (SimpleAnalysis) – the analysis object that will be used to query the bank.

  • series (Series) – the series of test used.

  • path (str) – the path to save the generated graphs.

  • show (bool) – do we try to show the graphs to the user directly, (need PyQt5).

  • extension (str) – format/file extension to use when saving the graphs.

  • limit (int) – nb max of run in the series to query (use sys.maxsize for no limit).

Return type:

None
